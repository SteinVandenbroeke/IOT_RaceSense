# IOT_RaceSense
IoT Telemetry and video for Amateur Motorsport

## Current problem


## Featurs
- Analyse your race line
- Pov camera's for the drivers
- Finish line camera
- Measure telemetry data from the kart (speed, acceleration, position on track)
- Auto generated highlight video of race (with map, pov's and raceline camera) using colision data

## AI model
A model trained to recognise every car, this is used to generate the highlight video.
